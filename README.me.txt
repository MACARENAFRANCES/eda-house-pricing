# Machine Learning & Artificial Intelligence Techniques (Economics Degree)
## Activity 1 – Exploratory Data Analysis (EDA)

**Uploaded:** 03 November 2025  

This repository contains an academic project developed as part of the course
*Machine Learning & Artificial Intelligence Techniques* within the Economics degree.

The objective of this activity is to perform an **Exploratory Data Analysis (EDA)**
before applying machine learning algorithms, in order to:

- Understand the structure and characteristics of the dataset
- Detect patterns and relationships between variables
- Identify the most relevant features for predictive modeling

---

## Workflow

1. Data loading and inspection  
2. Descriptive statistics of the dataset  
3. Missing values analysis and data cleaning  
4. Distribution analysis and outlier detection  
5. Correlation analysis between numerical variables  

---

## Dataset

The analysis is based on the dataset:

**house_pricing_train_Actividad1.csv**

This dataset contains information about residential properties and their sale prices,
including structural characteristics, zoning information, and sale conditions.

The target variable is **SalePrice**.
